Filings¶

Filings can be downloaded using the SECEdgar.base.Filing class found in the SECEdgar.filings module. These classes inherit from the SECEdgar.base._EDGARBase class. This class provides relevant **kwargs for making requests and the number of filings to fetch.

Filings¶

Note

SECEdgar tries to be flexible when interpreting what filing type you want. Filing types of the form Number-Letter such as 10-K can either be expressed as “10-k” or “10k” (case-insensitive).

Supported filing types include:

  • 10-K

  • 10-Q

  • 8-K

  • 13-F

  • 4

  • SD
